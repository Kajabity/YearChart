# Feature: Main Window

## Purpose

This is the application's main window.

## Requirements

- The Main Window is a Windows SDI Form layout.
- The application will follow current windows standards.
- A menu bar at the top with menu items added to support features when added.
- Initial menu item `File->Exit` will trigger application exit.
- A status bar at the bottom with panels for status display and action icons added to support features.
- The main view area will show the rendered chart with the current options, selected layout and view style.
