# Feature: About Box

## Purpose

Show clear version information to the user.

## Requirements

* The About box must display the current release version.
* The displayed release version must be derived from the same release version source used for the tagged build and installer.
* The displayed release version must match the installer release version.
* The About box may additionally display build metadata such as a build number or short git commit id.
* The about box is accessed via the main menu `Help->About`.
