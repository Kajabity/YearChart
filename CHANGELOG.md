# Changelog

All notable changes to YearChart are documented in this file.

This project uses semantic versioning from version `0.5.0` onwards.

## 0.5.0 - 2026-04-25

### Added

* Added GitHub Actions build automation.
* Added Dependabot configuration.
* Added vulnerability and secret scanning automation.
* Added release-versioning, About box, and engineering baseline specifications.
* Added this changelog.

### Changed

* Converted the release process to semantic versioning.
* Updated the project, build, and release workflows for .NET 10 LTS.
* Updated WiX and project versioning so release builds can derive version values from the release version.
* Updated copyright details.
